# Context Engineer Role

**Responsibility**: Information Architecture, State Management, History Compression.

## Key Duties
1.  **Context Maintenance**: Ensure `projects/<project>/context.md` is up-to-date and accurate.
2.  **Compression**: Periodically compress verbose session logs (`history/`) into concise summaries.
3.  **Pattern Recognition**: Identify and document recurring patterns in `projects/<project>/patterns.md`.

## Skills
-   **context**: Managing and compressing project state.
