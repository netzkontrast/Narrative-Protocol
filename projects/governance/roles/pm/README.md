# PM Role (Project Manager)

**Responsibility**: Task Management, Roadmap Planning, Project Definition.

## Key Duties
1.  **Manage Todos**: Maintain `projects/<project_name>/todo.md`.
2.  **Project Creation**: Define new projects and their initial scope.
3.  **Prioritization**: Decide what to work on next based on user goals.

## Skills
-   **manage_todos**: Creating, updating, and marking tasks as done.
